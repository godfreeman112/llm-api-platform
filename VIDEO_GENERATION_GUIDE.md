# 视频生成模型使用指南

## 🎬 快速开始

### 1. 添加 Seedance 2.0 模型

#### 步骤一：登录管理后台
```
访问: http://localhost:3000 (或您的部署地址)
账号: admin
密码: admin123456 (首次登录后请修改)
```

#### 步骤二：进入模型管理
点击左侧菜单 → **模型管理** → 点击右上角 **添加模型** 按钮

#### 步骤三：填写模型信息

**基本信息**
- 模型名称: `seedance-2.0-pro`
- 模型类型: ⭐ 选择 **视频生成模型**

**API配置**
- API端点: 
  ```
  https://ark.cn-beijing.volces.com/api/v3/videos/generations
  ```
- API密钥: `<从火山引擎获取的API Key>`

**视频参数**
- 视频时长: `5s` (建议先测试)
- 视频分辨率: `1080p` (高清)
- 输出格式: `MP4`

**描述**
```
火山引擎 Seedance 2.0 视频生成模型
支持文生视频、图生视频、多模态混合参考
```

#### 步骤四：保存并测试
点击 **确定** 保存 → 在列表中找到模型 → 点击 **测试** 验证连接

## ⚙️ 参数说明

### 视频时长 (videoDuration)

| 选项 | 适用场景 | 预估成本 |
|------|---------|---------|
| 5秒  | 测试、短视频片段 | ~5元 |
| 10秒 | 产品展示、简短动画 | ~10元 |
| 15秒 | 完整场景、短剧片段 | ~15元 |

**建议：** 先用5秒测试效果，满意后再使用更长时长

### 视频分辨率 (videoResolution)

| 选项 | 质量 | 速度 | 成本 |
|------|------|------|------|
| 720p | 良好 | 快 | 较低 |
| 1080p | 优秀 | 较慢 | 较高 |

**建议：** 
- 测试阶段使用 720p
- 正式生产使用 1080p

### 输出格式 (outputFormat)

| 格式 | 兼容性 | 文件大小 | 推荐场景 |
|------|--------|---------|---------|
| MP4  | 广泛支持 | 中等 | 通用场景 |
| WEBM | 现代浏览器 | 较小 | Web应用 |

**建议：** 优先选择 MP4 格式

## 🔧 高级配置

### API调用示例

#### Python
```python
import requests

def generate_video(prompt, duration="5s", resolution="1080p"):
    url = "http://localhost:3000/api/chat/send"
    
    headers = {
        "Authorization": "Bearer YOUR_TOKEN",
        "Content-Type": "application/json"
    }
    
    payload = {
        "model": 1,  # 模型ID，根据实际情况修改
        "message": prompt,
        "history": []
    }
    
    response = requests.post(url, json=payload, headers=headers)
    return response.json()

# 使用示例
result = generate_video(
    prompt="一只可爱的小猫在草地上玩耍",
    duration="5s",
    resolution="1080p"
)

print(result)
```

#### JavaScript
```javascript
async function generateVideo(prompt, duration = '5s', resolution = '1080p') {
  const response = await fetch('http://localhost:3000/api/chat/send', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${YOUR_TOKEN}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      model: 1,  // 模型ID
      message: prompt,
      history: []
    })
  });
  
  return await response.json();
}

// 使用示例
const result = await generateVideo('一只可爱的小猫在草地上玩耍');
console.log(result);
```

## ❓ 常见问题

### Q1: 视频生成需要多长时间？

**A:** 通常需要 30-60秒，具体取决于：
- 视频时长（5s vs 15s）
- 分辨率（720p vs 1080p）
- 服务器负载
- 网络状况

### Q2: 如何获得更好的视频质量？

**A:** 
1. **详细提示词**: 描述场景、动作、光影、镜头
2. **参考素材**: 提供图片或视频作为参考
3. **多次尝试**: 调整提示词直到满意
4. **高分辨率**: 使用 1080p 输出

### Q3: 视频生成失败怎么办？

**A:** 检查以下几点：
1. API密钥是否有效
2. 网络连接是否正常
3. 提示词是否合规（无敏感内容）
4. 配额是否充足
5. 查看错误日志定位问题

### Q4: 可以生成多长的视频？

**A:** 当前支持：
- 单次生成: 最长 15秒
- 更长视频: 可以分段生成后拼接

### Q5: 支持哪些语言？

**A:** 
- 提示词: 中文、英文均支持
- 建议: 使用英文提示词可能效果更好

示例:

```

示例:

```
[地点] + [时间] + [天气] + [特色元素] + [氛围]

### 工作流程

1. **构思阶段**
   - 明确视频主题和目标
   - 设计场景和镜头
   - 编写提示词草稿

2. **测试阶段**
   - 使用 5s + 720p 快速测试
   - 调整提示词直到满意
   - 记录成功的提示词

3. **生产阶段**
   - 使用 1080p 高质量输出
   - 根据需要选择时长
   - 批量生成提高效率

4. **后期处理**
   - 视频剪辑和拼接
   - 添加音效和音乐
   - 调色和特效
