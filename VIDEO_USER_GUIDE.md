# 🎬 视频生成使用指南

## 快速开始

### 方式一：通过Chat界面（推荐新手）

1. **登录系统**
   - 访问 http://localhost:8070
   - 使用您的账号登录

2. **选择视频模型**
   - 在首页点击任意视频生成模型
   - 或进入Chat页面，从顶部下拉菜单选择视频模型

3. **输入提示词**
   ```
   示例：一个年轻女性在樱花树下跳芭蕾舞，电影级画面，配合轻柔的钢琴曲
   ```

4. **发送消息**
   - 点击"发送"按钮或按 `Ctrl + Enter`

5. **查看任务状态**
   - 收到回复后，会看到一个任务卡片
   - 显示任务ID和当前状态

6. **查询生成结果**
   - 点击"查询生成结果"按钮
   - 系统会自动轮询任务状态（每5秒一次）
   - 进度条显示当前进度

7. **获取视频**
   - 生成成功后，视频会直接在对话中播放
   - 您可以：
     - 📥 点击下载视频
     - 🔗 点击复制链接
     - 👁️ 点击新窗口打开

### 方式二：通过专用页面（推荐高级用户）

1. **访问视频生成页面**
   - 在首页选择视频模型后自动跳转
   - 或直接访问 http://localhost:8070/video-generate

2. **配置参数**
   - **选择模型**: 从下拉列表选择
   - **提示词**: 详细描述您想要的视频内容
   - **参考图片** (可选): 上传一张图片作为参考
   - **视频时长**: 5秒 / 10秒 / 15秒
   - **视频分辨率**: 720p / 1080p

3. **生成视频**
   - 点击"生成视频"按钮
   - 等待2-5分钟（视复杂度而定）

4. **查看结果**
   - 视频会自动显示在页面下方
   - 可以点击"下载视频"保存

5. **查看历史**
   - 页面底部会显示您的生成历史
   - 可以快速回放之前的作品

## 💡 提示词技巧

### 好的提示词要素

1. **主体描述**
   ```
   ✅ 一个穿着红色连衣裙的女孩
   ❌ 一个人
   ```

2. **场景设定**
   ```
   ✅ 在巴黎埃菲尔铁塔前的广场上
   ❌ 在外面
   ```

3. **动作/行为**
   ```
   ✅ 优雅地旋转跳舞，裙摆飞扬
   ❌ 在跳舞
   ```

4. **视觉风格**
   ```
   ✅ 电影级画面，柔光效果，浅景深
   ❌ 好看的画面
   ```

5. **氛围/情绪**
   ```
   ✅ 浪漫温馨的氛围，夕阳余晖
   ❌ 美好的感觉
   ```

### 完整示例

```
✅ 优秀示例：
航拍视角，长城在清晨的云海中若隐若现，阳光穿透云层洒在古老的城墙上，
延时摄影风格，4K分辨率，史诗感，壮丽的自然景观

❌ 较差示例：
拍一下长城
```

## ⚙️ 参数说明

### 视频时长
- **5秒**: 适合短视频、GIF替代
- **10秒**: 平衡质量和速度（推荐）
- **15秒**: 适合复杂场景，生成时间较长

### 视频分辨率
- **720p**: 生成速度快，文件较小
- **1080p**: 高清画质，文件较大（推荐）

### 参考图片（图生视频）
- 支持 JPG、PNG 格式
- 文件大小不超过 20MB
- 可以让视频保持与参考图相似的风格

## 🔍 常见问题

### Q1: 视频生成需要多长时间？
**A**: 通常需要2-5分钟，具体取决于：
- 视频时长（越长越慢）
- 分辨率（1080p比720p慢）
- 服务器负载
- 网络状况

### Q2: 为什么显示"生成失败"？
**A**: 可能的原因：
- 提示词包含敏感内容
- API配额已用完
- 网络连接问题
- 服务器繁忙

**解决方法**:
- 检查提示词是否合规
- 联系管理员确认配额
- 稍后重试

### Q3: 如何保存生成的视频？
**A**: 有三种方式：
1. 点击"下载视频"按钮直接保存
2. 点击"复制链接"，然后粘贴到浏览器下载
3. 点击"新窗口打开"，在播放器中右键保存

### Q4: 视频链接失效了怎么办？
**A**: 火山方舟生成的视频URL有效期为24小时。建议：
- 生成后立即下载保存
- 如果链接失效，需要重新生成

### Q5: 可以同时生成多个视频吗？
**A**: 
- Chat界面：可以，每个对话独立
- VideoGenerate页面：建议一次生成一个，避免混淆

### Q6: 轮询时可以关闭页面吗？
**A**: 
- Chat界面：不建议，会中断轮询
- VideoGenerate页面：不建议，会丢失进度

**建议**: 如果需要长时间等待，可以使用VideoGenerate页面，它会自动管理任务状态。

## 🎯 最佳实践

### 1. 先测试再生成
- 先用简单的提示词测试效果
- 满意后再调整细节，生成高质量版本

### 2. 保存重要作品
- 视频URL只有24小时有效期
- 重要的视频请立即下载保存

### 3. 合理使用配额
- 视频生成消耗较多配额
- 建议先在脑海中构思好再生成

### 4. 利用历史记录
- VideoGenerate页面会保存历史
- 可以对比不同参数的效果

### 5. 网络环境
- 建议在稳定的WiFi环境下使用
- 避免在移动网络下生成大视频

## 📞 技术支持

1. 检查浏览器控制台是否有错误信息
2. 查看Docker容器日志：`docker-compose logs -f`
3. 联系系统管理员
