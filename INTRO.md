# 企业大模型API统一访问平台

## 🎯 项目简介

这是一个专为企业内部设计的大模型API统一管理平台，解决以下问题：

### 痛点
- ❌ 多个团队成员各自管理API密钥，混乱且不安全
- ❌ 无法统计每个成员的使用情况和费用
- ❌ 缺乏统一的配额控制和成本管理
- ❌ API密钥泄露风险高

### 解决方案
- ✅ 统一管理所有大模型API密钥
- ✅ 集中控制用户访问权限
- ✅ 实时监控使用情况和费用
- ✅ 灵活的配额管理机制

## 💡 核心功能

### 1. 统一认证
- 单点登录，统一身份认证
- JWT Token安全机制
- 角色权限控制（管理员/普通用户）

### 2. 多模型支持
支持主流大模型API：
- OpenAI (GPT-3.5, GPT-4)
- Anthropic (Claude)
- 百度文心一言
- 阿里通义千问
- 腾讯混元
- 自定义API

### 3. 使用监控
- 实时调用次数统计
- Token消耗追踪
- 费用计算和分摊
- 可视化图表展示

### 4. 配额管理
- 按用户设置使用配额
- 自动拦截超额使用
- 灵活调整策略

### 5. 对话测试
- 内置聊天界面
- 多模型切换对比
- 对话历史管理

## 🛠️ 技术架构

```
前端: Vue 3 + Element Plus + ECharts
后端: Node.js + Express
数据库: SQLite (轻量级，无需额外配置)
部署: Docker容器化
```

## 📊 应用场景

### 场景1: 研发团队共享API
- 问题: 10人团队都需要使用GPT API
- 方案: 统一购买企业API，通过平台分配配额
- 收益: 成本降低60%，管理效率提升

### 场景2: 多部门成本分摊
- 问题: 需要统计各部门的AI使用成本
- 方案: 为每个部门创建账户，设置配额
- 收益: 精确成本核算，预算可控

### 场景3: API密钥安全管理
- 问题: 担心员工离职带走API密钥
- 方案: 密钥集中管理，员工只通过平台使用
- 收益: 零泄露风险，随时撤销权限
